# Console

The Console is the browser UI for operating Agyn.

Use it for initial organization setup, agent review, app installation, devices, and operational inspection.

## Steps

1. Open the Console from your install URL.
2. Sign in with the configured OIDC provider.
3. Create or select an organization.
4. Review agents, runners, models, secrets, apps, and device enrollment areas.
5. Use role and membership controls to grant access intentionally.

## Expected outcome

Operators should be able to see organization-scoped resources and hand users a clear path to chat with available agents.

For automation, see [Terraform](./terraform.md).
