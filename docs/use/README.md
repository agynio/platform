# Use

This section is for end users and agent operators — the people who talk to agents, attach files, watch what agents do, see how much they cost, and access services exposed in agent workloads.

If you are configuring agents, models, or runners, see [Administer](../administer/README.md) instead.

## Where to start

If you are new to chat with agents, read these in order:

1. [Chat](./chat.md) — start and manage conversations.
2. [Files](./files.md) — attach files for agents to read.
3. [Run Timeline](./run-timeline.md) — see what the agent did.
4. [Reminders](./reminders.md) — let agents schedule follow-ups.
