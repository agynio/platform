# Deploy Your First Agent

This guide takes you from a prepared environment to a running agent.

## Steps

1. Install Agyn using your preferred platform deployment workflow.
2. Open the console and create an organization for your team.
3. Define an agent with a model, sandbox image, and idle timeout.
4. Attach the MCP tools the agent needs for its first task.
5. Set an initial budget so usage is visible and bounded.
6. Run a small test task and review the activity logs.
