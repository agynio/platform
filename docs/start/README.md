# Start

This section is the shortest path to a useful Agyn deployment.

You will prepare your environment, deploy the platform, and run an initial agent.

## Before you begin

- Confirm you can create infrastructure in your target environment.
- Decide which model provider and tools the first agent should use.
- Keep budget and access requirements close by for the initial configuration.

Continue with [Deploy your first agent](./deploy-your-first-agent.md).
