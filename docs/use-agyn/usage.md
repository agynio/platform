# Usage

Usage reporting helps teams understand activity and cost.

Metering tracks usage-related events, while Token Counting provides provider-aware token calculations.

## Steps

1. Start from the organization or agent usage view.
2. Review activity over the relevant time range.
3. Compare usage before and after model, prompt, or tool changes.
4. Use traces to inspect unusually expensive runs.
5. Adjust agent instructions, model choice, or tool design if usage is unexpected.

## Expected outcome

Operators should be able to connect model usage to agents, runs, and configuration changes.
