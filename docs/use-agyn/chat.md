# Chat

Chat is built on Threads.

Threads store messages and participants while Chat adds product behavior such as topics and unread counts.

## Steps

1. Open the chat surface.
2. Start a new conversation in the correct organization.
3. Add an available agent as an initial participant or later participant.
4. Send a message with clear instructions and any required files.
5. Watch for the agent response and follow-up questions.

## Expected outcome

A thread records the conversation between users, agents, and apps.

Agent availability and roles determine whether a user can start a conversation with a private agent.
