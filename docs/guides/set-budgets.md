# Set Budgets

Budgets help teams experiment with agents without losing cost control.

Apply limits early, then adjust them as usage patterns become clear.

## Recommended approach

1. Start with a conservative per-agent budget.
2. Monitor usage during the first real workflows.
3. Increase limits only after reviewing expected volume.
4. Separate production agents from experimental agents.
5. Revisit limits when model or tool configuration changes.

For configuration fields, see [Configuration](../reference/configuration.md).
