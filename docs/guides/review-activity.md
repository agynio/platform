# Review Activity

Activity review helps operators understand agent behavior and troubleshoot issues.

Use it after deployment, tool changes, or unexpected output.

## What to inspect

- Recent tasks and their completion status.
- Tool calls made through MCP integrations.
- Token and compute usage for the run.
- Errors from the sandbox or attached services.
- Whether the agent stopped after the configured idle timeout.

If something fails, continue to [Troubleshooting](../help/troubleshooting.md).
