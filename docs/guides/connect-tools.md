# Connect Tools

Tools let an agent work with systems such as files, databases, and SaaS APIs.

Start with the minimum set of tools required for the job.

## Workflow

1. Choose the MCP server that exposes the capability.
2. Configure credentials outside the model context.
3. Attach the MCP server to the target agent.
4. Run a small task that exercises the new tool.
5. Review logs to confirm the tool behaved as expected.

Read [Tools and MCPs](../concepts/tools-and-mcps.md) for the underlying model.
