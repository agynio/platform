# FAQ

## Is Agyn tied to one model provider?

No. Agyn is designed to be model-agnostic so teams can choose the right model for each agent.

## Where should tool credentials live?

Credentials should stay in platform-controlled configuration, not in prompts or model context.

## How should teams start?

Start with one narrow agent, a small tool set, and a visible budget.

Then expand after reviewing activity and operational fit.

For the first workflow, see [Deploy your first agent](../start/deploy-your-first-agent.md).
