# Platform UI Screenshot Guidelines

- Store Platform UI screenshots under `docs/images/platform-ui/`.
- Use kebab-case filenames that describe the screen and state, e.g. `llm-settings-credentials.png`.
- Prefer 1440px width captures in light mode unless a dark-mode scenario is required.
- Redact any tenant-specific or secret data before committing.
- Update corresponding PR descriptions to reference the latest captures.
