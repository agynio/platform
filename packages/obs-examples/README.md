# @hautech/obs-examples

Example scripts for Observability Stage 1.

Run (no Docker):
- Ensure @hautech/obs-server is running locally (PORT default 4319)
- Dev from sources: `pnpm --filter @hautech/obs-examples dev`
- Build + start: `pnpm --filter @hautech/obs-examples build && pnpm --filter @hautech/obs-examples start`

Env:
- `OBS_EXTENDED_ENDPOINT` (default: http://localhost:4319)
